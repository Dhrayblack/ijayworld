# IjayWorld 🌍

## overview

IjayWorld is a Flutter app made to help build social and community apps. It has a clean look, easy-to-use menus, and well-organized code. It's great for people who want to learn Flutter or start their own app project.

## features

- Easy to move between different screens
- Uses parts of code that can be reused in different places
- Ready to connect to a backend and add login features
- Works well on both Android and iPhone phones

## technology

- Flutter and Dart programming language
