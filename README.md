# IjayWorld 🌍

IjayWorld is a Flutter app designed for building social and community-driven platforms. With a clean, modern UI and a well-structured codebase, it’s perfect for developers who want to learn Flutter or kickstart their own app projects.

## features

- Easy to move between different screens
- Uses parts of code that can be reused in different places
- Ready to connect to a backend and add login features
- Works well on both Android and iPhone phones

## technology

- Flutter and Dart programming language
